### 循环结构
> while循环
```js
while(条件){
    条件为true执行里面的代码,条件为false退出循环
}
```

> do...while循环
```js
do{
    先执行一次里面的代码，然后进行条件判断，条件为true，执行代码，为false退出循环
}while(条件)
```

> for循环
```js
for(1初始值;2条件判断;4自身的改变){
    3要执行的代码
}
```

> 循环停止
+ break:终止循环
+ continue：停止本次循环

> 循环代码的四个条件
+ 有初始条件
+ 有条件判断
+ 有要执行的代码
+ 有自身的改变
